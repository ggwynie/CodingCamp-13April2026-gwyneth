# Spendly - Expense & Budget Visualizer

A simple and elegant expense tracking web application built with vanilla JavaScript.

## Features

- 💰 Track expenses with categories
- 📊 Visual spending breakdown with pie chart
- 🎨 Dark/Light mode toggle
- 🏷️ Custom category support
- 📈 Spending limit alerts
- 🔄 Sort transactions by date, amount, or category
- 💾 Local storage persistence

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Chart.js for data visualization
- Local Storage API

## How to Use

1. Open `index.html` in your browser
2. Add transactions with item name, amount, and category
3. Set a monthly spending limit (optional)
4. View your spending breakdown in the chart
5. Sort and manage your transactions

## Optional Features Implemented

✅ Custom categories
✅ Monthly summary view
✅ Sort transactions by amount/category
✅ Spending limit highlighting
✅ Dark/light mode toggle
